# Scrimble

Scrimble is a note-taking application that allows users to create and manage notes.

## Features

- Create new notes
- View existing notes
- Edit notes
- Delete notes

## Technologies Used

- NextJs
- Prisma
- PosgreSql
- Kinde Auth
